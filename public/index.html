HIOI

The HIOI (Home Investment Opportunity Identifier) project is an application that leverages the Zillow API to retrieve a list of houses based on the user's specified parameters. The application then utilizes Python to analyze the data and identify potential investment opportunities in the real estate market. By combining the power of web APIs and data analysis, HIOI aims to assist users [me] in making informed decisions when it comes to identifying investment opportunities in the housing market. Please note that this project is currently in development and more details will be available soon.
